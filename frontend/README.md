# Pride STEM Networking

This project is a React application designed to facilitate networking and collaboration within the STEM community. It provides a platform for users to connect, share resources, and engage in discussions.

## Features

- User authentication and management
- Interactive home page with resources and links
- Header and footer components for consistent layout
- Responsive design for accessibility on various devices

## Getting Started

To get a local copy up and running, follow these simple steps:

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/pride-stem-networking.git
   ```

2. Navigate to the frontend directory:
   ```bash
   cd pride-stem-networking/frontend
   ```

3. Install the dependencies:
   ```bash
   npm install
   ```

4. Start the development server:
   ```bash
   npm start
   ```
